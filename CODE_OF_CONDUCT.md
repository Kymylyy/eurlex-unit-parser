# Code of Conduct

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful and constructive
- Accepting feedback gracefully
- Focusing discussion on technical issues, not people

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks
- Trolling or intentionally disruptive behavior
- Publishing private information without permission

## Enforcement

Project maintainers are responsible for clarifying and enforcing these standards.
They may remove, edit, or reject comments, commits, code, and other contributions
that are not aligned with this Code of Conduct.

## Scope

This Code of Conduct applies in project spaces and public interactions
related to the project.
